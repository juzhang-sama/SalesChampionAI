销冠AI教练项目备份说明
========================

项目名称：销冠AI教练 (Sales Champion AI Coach)
备份时间：2025-07-27
项目状态：MVP版本完成

📁 备份内容：
├── src/                    # 源代码目录
│   ├── screens/           # 页面组件
│   ├── data/              # 数据文件
│   └── utils/             # 工具函数
├── App.tsx                # 应用入口文件
├── package.json           # 项目依赖配置
├── README.md              # 项目说明文档
├── SalesChampionAI_Fixed.apk  # 修复后的APK文件
└── local.properties       # Android SDK配置

🚀 核心功能：
1. AI沙盘话术推演 - 5个预设场景，实时评分
2. 销冠话术萃取 - 话术分析和模板生成
3. 管理驾驶舱 - 数据统计和风险管理
4. 设置页面 - 应用配置

🛠️ 技术栈：
- React Native 0.80.1
- TypeScript
- React Navigation
- React Native Gesture Handler

📱 安装说明：
1. 安装Node.js 18+
2. 安装React Native CLI
3. 配置Android Studio和SDK
4. 运行 npm install
5. 设置Android SDK路径
6. 运行 npx react-native start

🔧 构建说明：
- 开发版本：cd android && ./gradlew assembleDebug
- 发布版本：cd android && ./gradlew assembleRelease

📞 联系方式：
如有问题，请通过项目Issues联系。
