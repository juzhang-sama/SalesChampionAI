## 🔧 下次开发步骤

### 1. 环境准备
```bash
# 检查Node.js版本
node --version

# 安装依赖
npm install

# 设置Android SDK路径
echo "sdk.dir=F:\\Android" > android/local.properties
```

### 2. 启动开发
```bash
# 启动Metro服务器
npx react-native start

# 构建APK
cd android
./gradlew assembleDebug
```

### 3. 安装到设备
- 将 `android/app/build/outputs/apk/debug/app-debug.apk` 安装到手机
- 或使用 `adb install app-debug.apk`
